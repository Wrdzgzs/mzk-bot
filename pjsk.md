# 🎵 PJSK 相关功能

这里收录了《世界计划 多彩舞台》的相关查询指令。

## 1. 歌曲与谱面查询
| 指令 | 说明 | 示例 |
| :--- | :--- | :--- |
| **pjskinfo** / **song** | 查询歌曲详细信息 | `song 消失` |
| **pjskbpm** | 查看歌曲 BPM | `pjskbpm 消失` |
| **查bpm** | 查询指定 BPM 的歌曲 | `查bpm 200` |
| **谱面预览** | 查看谱面预览视频 (来源: ぷろせかもえ) | `谱面预览 消失 master` |

## 2. 玩家信息与绑定
> 支持查询 **日服(默认)**、台服(tw)、国服(cn)、韩服(kr)、国际服(en)。

*   **绑定账号**: `绑定 <UID>`
*   **查询个人**: `sk` (查看排名/分数) / `sk <UID>`
*   **生成名片**: `pjskprofile` (生成个人信息图)
*   **查房**: `查房 <排名>` (查询特定排名玩家信息)
*   **逮捕**: `逮捕 @某人` (查看群友的打歌情况)

## 3. MySekai (3D家园)
*   `ms分析`: 分析现存材料
*   `ms地图 <1-4>`: 查看资源分布图 (1:草原 2:花田 3:沙滩 4:废墟)
*   `ms照片 <序号>`: 下载游戏内拍摄的照片
