# <span class="gradient-text">Mizuki Bot 帮助文档</span>

**「 ボクは……ボクでいたいだけ (我只是……想做我自己) 」**

**Mizuki Bot** 是一款基于多端架构的综合娱乐机器人，形象灵感来源于 **晓山瑞希**。
我们专注于为 **Project Sekai (PJSK)** 和 **舞萌 DX (Maimai DX)** 玩家提供最便捷的查询与娱乐服务。

> 🎀 **开始使用**：点击左上角菜单，或直接查看下方功能卡片。

### 📢 常用指令一览

| 功能 | 指令示例 | 说明 |
| :--- | :--- | :--- |
| **PJSK** | `sk` / `pjskinfo 消失` | 查询分数 / 歌曲详情 |
| **Maimai** | `b40` / `锦标赛` | 查询 B40 成绩 / 牌子进度 |
| **娱乐** | `点歌 像鱼` / `今日词云` | 网易云点歌 / 群热词 |
| **表情** | `pjsk 瑞希 还是不知道` | 快速生成表情包 |
