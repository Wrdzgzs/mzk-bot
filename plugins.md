# 🧩 功能插件大全

这里收录了 Mizuki Bot 搭载的所有实用插件指令详解。

> 💡 **提示**：大部分指令支持模糊匹配，指令中的 `[]` 表示必填参数，`<>` 表示选填参数。

---

## 🎰 舞萌 DX 专项功能

### 1. MaimaiDX (查分与数据)
*核心查分插件，支持国服 DX 版本。*

| 常规指令 | 参数说明 | 功能描述 |
| :--- | :--- | :--- |
| **今日舞萌** | 无 | 查看今日舞萌运势 |
| **随个** | `<ver> <diff> [rating]` | 随机推荐一首歌曲/谱面 |
| **查歌** | `[title]` | 通过曲名文字查询歌曲 |
| **id** | `[id]` | 通过数字 ID 查询歌曲信息 |
| **是什么歌** | `[alias]` | 查询这个别名对应哪首歌 |
| **定数查歌** | `[min] [max]` | 查询指定定数范围内的歌 |
| **分数线** | `[diff+id] [score]` | 查询达成目标分数需要多少分 |
| **b50** | `<@某人>` | **查询 B50 成绩图** (最常用!) |
| **完成表** | `[level] <grade>` | 查看指定等级的通关/评价完成度 |
| **进度** | `[rating][grade]<plan>进度` | 查看牌子或等级的完成进度 (如: `13+SSS进度`) |
| **猜歌** | `mai猜歌` | 开启猜歌小游戏 |

> **参数图例**：
> *   `diff`: 难度 (绿/黄/红/紫/白)
> *   `ver`: 版本 (dx/标准)
> *   `rating`: 乐曲定数

### 2. Mai Arcade (机厅与排卡)
*机厅排队助手，解决排卡纠纷的神器。*

#### 👥 玩家排卡
*   `上机`: 将排在第一位的移至最后（轮到你了）
*   `排卡`: 加入当前机厅的队列
*   `退勤`: 离开队列
*   `排卡现状`: 查看当前谁在排队
*   `延后`: 有事离开，将自己位置往后挪一位

#### 🏠 机厅查询
*   `机厅人数`: 查看当前机厅有多少人
*   `机厅列表`: 查看已注册的机厅
*   `机厅地图 <机厅名>`: 查看机厅具体位置图

#### 🛠 管理员指令
*   `机厅人数+1` / `机厅人数-1`: 手动调整人数
*   `闭店`: 清空所有排队信息
*   `添加机厅 <名字>`: 注册新机厅

### 3. MaimaidX (移植版查分)
*   **指令**: `帮助maimaidx`
*   **功能**: 移植自 mai-bot 开源项目，提供更多维度的查分功能。

### 4. Partner Join (旅行伙伴)
*   **指令**: `加入帮助` / `join help`
*   **功能**: 生成类似“旅行伙伴加入”的图片（可把图片嵌入校徽圆框）。

## 🛠️ 群管与实用工具

### 1. 自动点赞订阅赞
*   **点赞**: 发送 `点赞`，Bot 给你点个赞。
*   **订阅赞**: 发送 `订阅赞`，Bot 每天中午 12 点自动为你点赞。

### 2. Analysis Bilibili (B站解析)
*   **用法**: 直接在群里发送 Bilibili 的视频/动态链接。
*   **效果**: Bot 会自动解析并发送视频封面、标题、简介等预览信息。

### 3. Steam 游戏状态
*   **绑定**: `steam绑定 [SteamID]` (需先获取64位ID)
*   **查询**: `steam状态`
*   **功能**: 播报群友正在玩的 Steam 游戏状态，支持 `steam列表` 查看群友都在玩啥。

### 4. BlueArchive Title (BA标题生成)
*   **指令**: `batitle [左边文字] [右边文字]`
*   **示例**: `batitle Mizuki Bot`
*   **效果**: 生成蔚蓝档案风格的 Logo 图片。

### 5. PicStatus (运行状态)
*   **指令**: `运行状态` / `status` / `zt`
*   **效果**: 发送一张包含 CPU、内存、硬盘占用率的精美图片。

### 6. 发言统计
*   **指令**: `发言统计`
*   **效果**: 统计并展示今日群聊的“龙王”排行榜。

### 7. PicMenu Next (图片菜单)
*   **指令**: `帮助`
*   **效果**: 生成一张包含所有插件功能的图片菜单（即你之前看到的那张紫色的图）。

## 🎪 娱乐与小游戏

### 1. Songpicker2 (点歌)
*   **指令**: `点歌 [歌名]`
*   **示例**: `点歌 像鱼`
*   **来源**: 网易云音乐，支持热评显示。

### 2. 词云 (Word Cloud)
*   **今日词云**: `今日词云` - 生成今天群里讨论最多的关键词图片。
*   **昨日词云**: `昨日词云` - 回顾昨天的热门话题。
*   **我的词云**: `我的今日词云` - 看看你今天最爱说什么词。

### 3. Playercheck (成分查询)
*   **查询**: `cf.list` (查看支持的游戏)
*   **添加**: `cf.add [游戏名]` (给自己贴标签)
*   **功能**: 查询群友都玩什么音游，寻找同好。

### 4. 喜(悲)报生成器
*   **喜报**: `/喜报 [文字]` -> 生成喜庆的红底通告。
*   **悲报**: `/悲报 [文字]` -> 生成黑白风格的通告。

### 5. 选择困难症
*   **指令**: `选 [选项A] 选 [选项B]`
*   **示例**: `选 肯德基 选 麦当劳`
*   **效果**: Bot 会帮你随机挑一个。
