# 🧩 通用功能插件

mzk-bot 整合了多个实用的 NoneBot 插件。

| 插件名称 | 功能简介 | 指令示例 |
| :--- | :--- | :--- |
| **Analysis Bilibili** | B站链接自动解析 | (直接发链接) |
| **MC QQ** | Minecraft 服务器互通 | (需配置) |
| **MCSM 小助手** | 管理 MC 服务器面板 | `mcsm help` |
| **Mai Arcade** | 舞萌机厅排卡、查线 | `排卡` |
| **Maimaid x** | 舞萌 DX 查分 | `b40` |
| **Meme Stickers** | PJSK 表情包生成 | [查看详细教程](meme-stickers.md) |
| **PicStatus** | 服务器运行状态图 | `status` |
| **Songpicker2** | 网易云点歌 | `点歌 歌曲名` |
| **Steam游戏状态** | 播报好友Steam状态 | (自动) |
| **词云** | 生成群聊词云 | `今日词云` |
| **发言统计** | 统计群友发言数 | `发言统计` |
| **喜报/悲报** | 生成黑白风格通告 | `喜报 xxx` |
| **选择困难症** | 帮你在多个选项中抉择 | `bot帮我选 A B C` |
